# skills/cost-models

Cost modeling templates and pricing data for common scenarios.

## What belongs here

- Provider price sheets (updated quarterly)
- Pre-built cost models for common workload types
- TCO calculation templates
- Egress/transfer cost calculators

## File format

```markdown
# Cost Model: [Scenario]

## Workload profile
## DO pricing
## Competitor pricing
## Key cost drivers
## Common surprises
## Calculation template
```

## Pricing update cadence

Update pricing files at least quarterly. Tag with `updated: YYYY-MM-DD`.
