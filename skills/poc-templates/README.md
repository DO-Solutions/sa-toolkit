# skills/poc-templates

POC runbooks and validation templates for specific products and use cases.

## What belongs here

- Product-specific POC runbooks (DOKS, Managed DBs, App Platform, GPU)
- Load testing scripts and configurations
- Validation checklists by workload type
- Post-POC findings templates

## File format

```markdown
# POC Template: [Product / Use Case]

## Prerequisites
## Environment setup
## Validation steps
## Success criteria benchmarks
## Common failure modes
## Cleanup checklist
```
