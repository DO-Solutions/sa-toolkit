# skills/qualification

Qualification frameworks, scoring rubrics, and discovery question banks.

## What belongs here

- Industry-specific qualification guides (fintech, gaming, SaaS, e-commerce)
- Red flag patterns (workloads to qualify out)
- Discovery question templates by use case
- Competitive displacement playbooks

## File format

```markdown
# Qualification: [Topic]

## Ideal customer profile signals
## Disqualifiers
## Key discovery questions
## Scoring adjustments
## Example engagements
```
